웹폰트

webfont - 링크, 도메인, 브라우저와 상관없이 웹페이지내에서 폰트가 연결되어지는 것
내 컴퓨터 내에 설치된 폰트를 주로 이용함

webfont-cdn : 웹에서 폰트를 긁어다 와서 전달하는 방법
다만 출처가 되는 웹페이지가 느려지거나 터질 경우 폰트가 나타나지 않는 경우가 생길 수 있으니 주의 할 것

@font-face :폰트 파일을 다운받거나 가져와 문서와 직접 연결하여 폰트를 선언하는 방식
오류가 적고 적용이 빠름
폰트를 직접 구해와야 한다는 점이 번거로움
EOT : IE8 이하
TTF : 가장 흔히 쓰이는 압축을 푸는 형태로 사용
WOFF : 대부분의 브라우저에서 지원
WOFF2 : WOFF보다 압축률이 좋음
